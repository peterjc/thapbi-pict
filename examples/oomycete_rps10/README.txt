This example is based on:

    Foster *et al.* (2021) A new oomycete metabarcoding method using the rps10 gene
    https://doi.org/10.1094/PBIOMES-02-22-0009-R
    https://doi.org/10.1101/2021.09.22.460084 (preprint)
    https://github.com/grunwaldlab/OomyceteDB
    https://www.ebi.ac.uk/ena/browser/view/PRJNA699663

It will download 64 raw FASTQ files (24 pairs), taking about 2.3GB on disk.
