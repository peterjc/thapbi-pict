.. _fecal_sequel:

Bat Mock Community COI
======================

This example considers mock communities of 3 bat species (in different ratios)
using the COI marker, using one of the amplicon sequencing libraries from:

    Walker *et al.* (2019)
    A fecal sequel: Testing the limits of a genetic assay for bat species
    identification.
    https://doi.org/10.1371/journal.pone.0224969

The example highlights the importance of good database coverage with the
default ``onebp`` classifier method.
